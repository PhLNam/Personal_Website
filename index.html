Battleship Game

This project outlines the development of a console-based Battleship game in C++. Designed as an offline, single-player experience, the game pits the player against a professionally trained computer. Players will have the flexibility to customize their gameplay by selecting the bot's difficulty level and defining the size of the game board. Furthermore, the game will allow players to strategically place various types of ships on their grid before the battle commences. The game will be implemented with a strong focus on object-oriented programming (OOP) principles, aiming for a clear, modular structure and extensibility. This initial console version serves as a foundational learning exercise in C++, with future plans to enhance the game with a graphical user interface and more advanced features as C++ mastery is achieved.
